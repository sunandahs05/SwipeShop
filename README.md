# SwipeShop 🛒

SwipeShop is a swipe-based e-commerce platform designed to reduce decision fatigue by allowing users to discover products using swipe gestures.

## 🔧 Tech Stack

* Backend: Python (Flask) *(upcoming)*
* Database: MySQL 8.0
* Frontend: HTML, CSS, JavaScript *(upcoming)*

## 📂 Current Status

* ✅ Database Schema (3NF)
* ✅ Stored Procedures
* ✅ Triggers
* ⏳ Backend (Flask)
* ⏳ Frontend

## 🚀 Features

* Swipe-based product discovery
* Personalized recommendations
* Cart + Atomic Checkout (Transactions)
* Order tracking
* Admin analytics & reports
